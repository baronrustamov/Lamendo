## Overview

Lamendo is an online bulletin board system (BBS) made using Flask and SQLite.

Check out an instance I host at [https://www.lamendo.xyz/](https://www.lamendo.xyz/).

<br>


## Getting Started

`pip install -r requirements.txt`

`cd app`

`py main.py`
